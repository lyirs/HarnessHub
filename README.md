# AIHarnessHub

A living registry of open-source harness engineering projects for AI agents, coding workflows, and evaluation pipelines.

[中文说明](README_ZH.md)

## Snapshot

- 4 tracked areas
- 42 indexed GitHub repositories
- GitHub star and link snapshot: 2026-04-12

## Inclusion Rules

- Public GitHub repositories with explicit relevance to AI or agent harness engineering.
- GitHub star threshold: 40+ stars as of 2026-04-12.
- Includes learning guides, harness runtimes, workflow kits, and evaluation harnesses.
- Excludes unrelated CI/CD `Harness` products, generic test harnesses, and non-AI projects.

## Notes

- Star counts are point-in-time GitHub API snapshots and will drift.
- This project is an index of upstream repositories; it does not mirror source code or benchmark assets.
- Some repositories blend docs, skills, templates, and runnable systems. They are grouped by their primary value for practitioners.
